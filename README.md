# About
This project run on ReactJs v18 on VITE framework
CRM_REACT contains:
* components
* json server
* react router dom
* tailwind
---
# Dependencies

* Tailwind
* json server
* react router dom

# How to run

    $ git clone <this_repo>
    $ cd client_admin/crm_react
    $ npm i 
    $ npm run dev
